# parallel_pagerank
A multithreaded application of the PageRank algorithm

## Specifications 
> Input : 
>> The input file is provided through the command line arguments. Lines starting 
>> with '#' are treated as comments.

> Output :
>> The output has the same name as the input file with the extension .result.

> Running on multiple threads :
>> The application can run on multiple threads by changing the defined macro 
>> THREAD at the top of the source code and recompiling.
